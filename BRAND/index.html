Accessibility

Contrast guide

Every combination is verified against WCAG 2.1 thresholds. Follow the recommended column. Reference: WCAG 2.1 Contrast.

Background Foreground Ratio Body text
4.5:1		 Large text
3:1		 UI / Graphics
3:1		 Usage
Sand Teal Ink #1F3536 10.1:1 AAA AAA AAA DEFAULT
Sand Teal Deep #2A5A5C 5.9:1 AA AAA AAA Eyebrows, stats, secondary text
Sand Teal #3F8487 3.3:1 NO OK OK Headings, italics, links (large only)
Sand Clay #C75A2A 3.2:1 NO OK OK Decorative accents only; not body text
Sand Clay Deep #9E441F 4.9:1 AA AAA AAA Small accent text, hover states
Teal Sand 90% 3.0:1 NO OK OK Slab backgrounds (large text only)
Clay Sand #ECE0CC 3.2:1 NO OK OK CTA buttons (bold, uppercase text)
Teal Ink Sand 90% 8.2:1 AAA AAA AAA Pill buttons, mobile nav, badges

Key rules

  • Body text: always Teal Ink (#1F3536) on Sand. Ratio 10.1:1 passes AAA.
  • Teal (#3F8487) and Clay (#C75A2A) fail AA for body text on Sand. Use only on large headings or UI elements.
  • Clay buttons pass AA for large/UI text (3:1) with bold, uppercase, and sufficient padding.
  • Teal slab backgrounds require light text at large sizes. Do not use Teal or Clay for body text on any background.
  • Every combination on this site was chosen with accessibility in mind. All body text passes WCAG AA or AAA.
Proper Usage

Brand guidelines

These rules protect the integrity and recognizability of the Tellus brand. Follow them whenever you use our name, logo, or colors.

Do

  • Use the official logo files from this page. Never recreate or trace the wordmark.
  • Keep a clear exclusion zone around the logo equal to the height of the letter "T".
  • Use the dark logo on light backgrounds and the white logo on dark backgrounds.
  • Follow the contrast guide. Body text must be Teal Ink (#1F3536) on Sand (#ECE0CC).
  • Use the brand typefaces: Fraunces for headings, Inter for body, JetBrains Mono for data.
  • Write "Tellus Cooperative" or "Tellus Cooperative Foundation" on first mention.
  • Link back to telluscoop.org when using our brand assets online.

Don't

  • Stretch, compress, rotate, or distort the logo in any way.
  • Change the logo colors. Use the approved variants only.
  • Add drop shadows, outlines, borders, gradients, or effects to the logo.
  • Place the logo on busy, low contrast, or clashing backgrounds.
  • Use Clay (#C75A2A) or Teal (#3F8487) for body text. They fail WCAG AA.
  • Create your own lockups, sub-brands, or derivative logos without written permission.
  • Use the Tellus brand to imply endorsement, partnership, or sponsorship without consent.

Clear space and minimum size

The logo must be surrounded by a clear space equal to the height of the letter T in the wordmark. No other elements may enter this zone.

Minimum print size: 30 mm width. Minimum digital size: 120 px width.
